# react-native-wf-common-components
&emsp;ReactNative通用组件：在ReactNative项目开发过程中，经常会碰到一些非常通用的组件，所以这里封装出来使用。

----
### AnimatedModal
其实就是一个Modal，但是对react-native提供的Modal进行了封装。
加入了动画效果、内容位置等功能，使展示效果更完美。

**支持的动画效果**：
- 淡入/淡出
- 左进/左出
- 上进/上出
- 右进/右出
- 下进/下出

**支持的内容位置**：
- 上、下、左、右、中间
